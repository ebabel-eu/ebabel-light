# ebabel-light
Setup a THREE.js PointLight and add it to the scene.

## Instructions for Module Usage in your project/game

### Install
```
npm i --save ebabel-light
```

### Usage
```
const light = require('ebabel-light');

light({ THREE, scene });
```
